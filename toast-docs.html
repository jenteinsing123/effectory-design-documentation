Deprecated

Toast

Toast is deprecated. Use a System Notification instead — it's the same component, and it covers transient messages across the product.

Don't use Toast for new work. A toast was a transient message shown bottom-centre. In practice it's the same underlying component as the System Notification, only positioned differently. To keep one pattern, route transient feedback through the System Notification (top-right) and reserve in-page feedback for the Inline notification.

What to use instead

→
System Notification. The outcome of an action or a background event — "Export ready", "Sync failed". Appears top-right, can carry an action, errors persist. See System Notifications →
→
Inline notification. Feedback tied to a specific element or section on the page. See Inline notification →

Already using a toast?

Existing toasts keep working, but plan to migrate them to a System Notification. The notification component and service are the same; only the placement differs (bottom-centre → top-right).
