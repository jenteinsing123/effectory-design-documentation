Loading

An animated spinner that signals an indeterminate wait — when the system is busy and there's no progress to report. For content that's loading into a known layout use a Skeleton; for a measurable task use a Progress Bar.

Anatomy

A circular track with a coloured arc that rotates continuously.

Loading
1
Track — The faint full ring the arc travels along.
2
Arc — The solid segment that rotates to show activity.
3
Label — Optional "Loading" text below the spinner (block loader only).

Usage

Reach for a spinner when a wait is short and you can't show progress. When you know the shape of what's coming, a Skeleton feels faster; when you can measure progress, use a Progress Bar.

Indeterminate, short waits

Loading
Do
A spinner for a brief, unmeasurable wait.
55%
Don't
Use a spinner when you can show real progress — that's a Progress Bar.

Variants

Two forms — a centred block loader with a label, and a compact inline loader — each available in three sizes.

Form

Loading
Block loader — A centred spinner with an optional "Loading" label. Use it to fill an area or a whole view while it loads.
Saving…
Inline loader — A small spinner that sits in a line of text or inside a button while an action runs.

Size

Small — 16px — Inside buttons, table rows, or next to a single line of text.
Medium — 24px — The default, for cards and panels.
Large — 32px — Full-page or large-area loading.

Content guidelines

A label is optional — when you add one, keep it short and honest.

Say what's happening, briefly

Use a short verb or just "Loading". Don't promise a duration you can't keep.

Accessibility

A spinner is decorative on its own — make the wait perceivable to everyone.

→
Announce it. Put role="status" on the loader (or its region) so assistive tech announces that content is loading.
→
Label icon-only spinners. A spinner with no visible label needs aria-label="Loading".
→
Remove it when done. Take the loader out of the DOM (or update the region) once loaded, so it doesn't keep announcing.
→
Don't trap focus. A spinner isn't interactive — keep it out of the tab order.

Accessibility

Make the loading state announce itself.

→
Status role. Wrap with role="status" (the eff-loader handles this) so screen readers hear the change.
→
Label. Add aria-label="Loading" when there's no visible text.
→
Reduced motion. The spin is the only signal — keep it, but avoid pairing with other large motion.
