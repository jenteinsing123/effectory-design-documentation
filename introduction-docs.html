Getting started

Three steps to get up to speed, wherever you're starting from.

1
Read the Design Principles
The Design Principles explain the values and trade-offs that inform every decision — and act as the tiebreaker when two valid options exist.
2
Learn the token system
Spend 10 minutes with Colors and Spacing. Understanding how semantic tokens map to raw values makes every component page much faster to read.
3
Open any component page
Each page has a Design tab (anatomy, usage, variants, states, tokens) and a Developer tab (live playground, Angular API, copy-ready code). A ⚠ Figma gap callout marks anything not yet built.
