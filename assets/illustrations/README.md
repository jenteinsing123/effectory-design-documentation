# Dialog-illustraties

De announcement-dialog (`prototypes/announcement-playground.html`) verwacht hier vijf PNG's.
Sla de afbeeldingen op met exact deze namen:

| Bestand | Illustratie | Slide | Kleur |
|---|---|---|---|
| `whatsnew.png` | "What's New" cover-graphic (transparant) | Intro (eerste slide) | geel |
| `notifications.png` | HR-notificatiegrid | Stap 1 | paars |
| `tasks.png` | Onboarding Taken (tabel) | Stap 2 | rood |
| `inbox.png` | Inbox-hoogtepunten | Stap 3 | groen |
| `welcome.png` | Welkom Anneke (onboarding-kaarten) | Stap 4 | oranje |

- De **intro** toont `whatsnew.png` met confetti + sparkles eromheen (transparante delen laten de feestelijke achtergrond doorschijnen).
- Ontbreekt een bestand, dan valt die slide terug op het placeholder-icoon + accent-blob.
- Aanbevolen formaat: ~600×440 (mediacontainer is 600px breed × 440px hoog, `object-fit: cover`).
