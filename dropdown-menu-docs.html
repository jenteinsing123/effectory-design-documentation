Dropdown Menu

A floating list of actions or options, opened from a trigger. Use it for a compact set of commands or choices that don't need to stay on screen.

Anatomy

A menu is a floating surface holding rows. Each row has a label, optionally an icon or avatar, and an optional trailing element; an optional header and group labels organise longer lists.

Usage

Use a dropdown menu for a short list of actions or options tied to a trigger. For a few always-visible options use a Segmented control or Tabs; for a long, searchable list use a Select.

Content guidelines

Menu items are scanned quickly — keep labels short and action-oriented.

For a menu of commands, lead with the verb ("Rename", "Duplicate"). For a choice menu, use plain nouns.

Accessibility

A menu must be operable and announced as a menu.

→
Roles. The surface is role="menu", items role="menuitem" (or menuitemcheckbox for selectable ones).
→
Keyboard. Open with Enter/Space/arrow, move with arrows, choose with Enter, close with Escape — focus returns to the trigger.
→
Trigger link. The trigger has aria-haspopup="menu" and aria-expanded reflecting open state.
→
Icon-only items. Always include a text label; don't rely on the icon alone.

Usage

The dropdown is Angular Material's mat-menu, opened from a trigger with matMenuTriggerFor. Items are button mat-menu-item (or a mat-menu-item for links).

Basic

<button mat-button [matMenuTriggerFor]="menu">Open dropdown</button>

<mat-menu #menu="matMenu">
  <button mat-menu-item disableRipple>
    <eff-mat-icon class="icon" [iconName]="'user'"/>
    <span>Open profile</span>
  </button>
  <button mat-menu-item disableRipple>
    <eff-mat-icon class="icon" [iconName]="'log-out'"/>
    <span>Log out</span>
  </button>
</mat-menu>

Link items

Use an anchor with mat-menu-item and the icon-before class for items that navigate.

<a mat-menu-item class="icon-before" href="https://support.effectory.com" target="_blank" rel="noopener noreferrer" disableRipple>
  <eff-mat-icon class="extra-small-icon icon" [iconName]="'external-link'"/>
  <span>Open support</span>
</a>

Submenu

A submenu item shows a trailing chevron; on hover it opens a nested menu beside the parent. In Angular Material, point the item's matMenuTriggerFor at a second mat-menu.

Share
Copy link
Email
<mat-menu #menu="matMenu">
  <button mat-menu-item disableRipple>
    <eff-mat-icon class="icon" [iconName]="'user'"/><span>Profile</span>
  </button>
  <!-- opens the nested menu -->
  <button mat-menu-item [matMenuTriggerFor]="share" disableRipple>
    <eff-mat-icon class="icon" [iconName]="'share'"/><span>Share</span>
  </button>
</mat-menu>

<mat-menu #share="matMenu">

Accessibility

mat-menu wires up the menu roles and keyboard model; keep these in mind.

→
Roles + keyboard. Material sets menu/menuitem, aria-haspopup, arrow-key navigation and Escape-to-close by default.
→
Labels. Every item needs visible text; an icon alone isn't enough.
→
Links vs buttons. Use a mat-menu-item for navigation, button mat-menu-item for actions.
