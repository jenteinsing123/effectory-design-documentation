Shadows

Shadows lift an element off the page. Engage uses eight tokens — each scoped to a specific surface — built from a single navy tint at varying opacity, offset, and blur. Reach for the token that matches the surface, not the visual weight you want.

✨ Guidance

Match the token to the surface, not the weight

Each token is named after what it wraps — card, dialog, popover. Pick the one that describes the element you're styling, not the one that "looks about right." This keeps elevation consistent across every product surface.

Pair resting and hover states

Interactive elements move between two tokens: action → action-hover for buttons, card → card-hover for tiles. Don't leap across unrelated tokens on hover — the jump feels unmotivated.

One layer at a time

Nested shadows stack noise and blur the hierarchy. A dialog already casts shadow/dialogs — cards inside it should sit flat. Shadow is a scarce signal; use it only where the surface truly leaves the page.
