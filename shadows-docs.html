Every shadow uses the same base tint: navy #192743. Only opacity, offset, and blur change — so shadows always feel part of the same system regardless of which token you use.

Guidance

Match the token to the surface, not the weight

Each token is named after what it wraps — card, dialog, popover. Pick the one that describes the element you're styling, not the one that "looks about right." This keeps elevation consistent across every product surface.

Pair resting and hover tokens

Interactive elements transition between two tokens: action → action-hover for buttons, card → card-hover for tiles. Don't jump across unrelated tokens on hover — the depth shift feels unmotivated.

One layer at a time

Nested shadows create visual noise and blur hierarchy. A dialog already casts shadow/dialogs — cards inside it should sit flat. Shadow is a scarce signal; use it only where the surface truly leaves the page.
